# mFD
This project will allow me to practice obtaining data from KSP and creating data visualizations using python libraries such as numpy, matplotlib or pandas.

## V1
In V1, the v1.py script interfaces with kRPC and records flight data until the aircraft flown touches the ground. At this point, the data is written to flightdata.csv The v1vis.py produces a chart of the relevant flight parameters. Here's a sample graph produced:
